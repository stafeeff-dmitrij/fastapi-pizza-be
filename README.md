# FastAPI Pizza

FE - https://github.com/stafeeff-dmitrij/react-pizza-fe

Запуск приложения (с автоматическим импортом фикстур)
```
docker compose up -d
```

Ручной импорт фикстур (опционально)
```
docker exec -it fastapi-pizza-be python -m fixtures.import
```
